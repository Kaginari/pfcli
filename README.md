## 

##### Init go module :
```
 $ go mod init gitlab.com/XXXXX
```
##### Run go project : 

```
$ go run main.go
```

##### Install go binary :

```
$ go install
```

##### lunch binary :

```
$ pfcli <COMMAND> <ARG>
```
